# 🧮 BMI Calculator (Калькулятор ІМТ)

Простий та зручний десктопний застосунок для розрахунку **Індексу маси тіла (ІМТ)**, написаний на мові Python з використанням бібліотеки `tkinter`.

Ми вітаємо внесок спільноти! Якщо ви знайшли баг або хочете запропонувати нову функцію:
1. Ознайомтеся з нашою інструкцією: [**Документація для розробників (CONTRIBUTING.md)**](CONTRIBUTING.md).
2. Створіть "Issue" або надішліть "Pull Request".

## 📌 Основні можливості
* **Миттєвий розрахунок**: Введіть вагу та зріст, щоб отримати результат.
* **Категоризація**: Автоматичне визначення категорії (Недостатня вага, Норма, Надмірна вага).
* **Валідація даних**: Захист від введення некоректних символів або нульових значень.
* **Автоматизоване тестування**: Весь математичний функціонал покритий тестами `unittest` та `pytest`.
