
# 👩‍💻 Правила розробки (Contributing Guidelines)

Дякуємо, що вирішили допомогти покращити Калькулятор ІМТ! Ось короткий посібник про те, як почати.

## 📝 Як запропонувати зміни?
1. **Fork**: Створіть власну копію репозиторію.
2. **Branch**: Створіть нову гілку для своєї фічі (`git checkout -b feature/CoolNewFeature`).
3. **Commit**: Опишіть свої зміни чітко (`git commit -m 'Додано нову функцію...'`).
4. **Push**: Завантажте гілку в свій репозиторій.
5. **Pull Request**: Надішліть запит на злиття в основну гілку.

## 🧪 Вимоги до тестування
Ваші зміни не повинні ламати існуючий функціонал. Обов'язково запускайте тести перед відправкою:

# Запуск усіх тестів pytest
pytest

# Запуск тестів unittest
python -m unittest test_bmi.py
